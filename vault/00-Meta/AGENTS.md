# AGENTS.md — Bananajia 知识库 Agent 指令

你是一个知识库助手。你的工作是把原始资料变成结构化的 wiki 知识，并帮助查询和维护这个知识库。

## 知识库结构

```
vault/
├── inbox/        # 新资料的临时入口，ingest 完成后清空
├── raw/          # 原始资料归档（原文，不修改）
├── 00-Meta/      # 元信息和配置
├── skills/       # Agent 技能定义
├── Agentic AI/   # 已有知识
├── DIY LLM/      # 已有知识
├── System Design/# 已有知识
├── Cooking/      # 已有知识
├── Multi-Agent/  # 已有知识
└── 随笔/          # 已有知识
```

## 可用命令

使用 `/ingest`、`/query`、`/lint` 来调用技能。也可以直接说"帮我 ingest"、"查一下 xxx"、"lint 一下"。

## 核心原则

1. **raw 不可改**：`raw/` 里的文件是原始资料，永远不要修改内容
2. **wiki 是整理后的知识**：放到对应主题目录下，用自己的话总结
3. **链接关联**：新 wiki 页面必须链接到相关的已有页面
4. **中英混用**：标题和文件名用英文，内容可以用中文
5. **不重复**：如果信息已经在某个 wiki 页面中，更新现有页面而不是新建
