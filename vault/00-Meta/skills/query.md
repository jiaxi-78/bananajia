# Skill: Query

## 触发方式
- `/query <问题>` — 查询知识库
- 直接提问也可以自动触发

## 流程

### 1. 搜索 wiki 页面
- 根据问题关键词，搜索 vault 中所有 `.md` 文件
- 找到最相关的 wiki 页面（优先 `wiki/` 和主题目录）
- 忽略 `raw/`、`inbox/`、`00-Meta/skills/` 中的内容

### 2. 综合回答
- 基于找到的 wiki 页面综合回答
- 如果 wiki 信息不足，去 `raw/` 查找原始资料补充
- 每个关键信息注明来源页面：`[来源：[[页面名]]]`

### 3. 回答格式
```
## 回答
综合回答内容...

## 来源
- [[相关页面A]] — 相关片段
- [[相关页面B]] — 相关片段
```

### 4. 发现知识缺口
- 如果某些信息知识库中没有，明确告知
- 建议用户将相关资料放入 `inbox/` 进行 ingest
