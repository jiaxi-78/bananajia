# System Design Interview 课程化学习笔记

这是一份基于《System Design Interview: An Insider's Guide》整理出的短 lesson 课程站点。目标不是复刻原书，而是把系统设计面试的知识拆成更短、更连续、更适合练习的学习路径。

## 学习方式

每一节都控制在较短篇幅内，并固定包含：

- `本节目标`：先说明这一节学什么。
- `先说结论`：快速抓住核心。
- `拆开理解`：把概念分成小块。
- `一个例子`：用场景或类比帮助吸收。
- `小结`：收束重点。
- `思考题`：检查理解并引向应用。
- `下一节预告`：告诉你下一步会接到哪里。

## 课程主线

1. 从扩展性和估算建立系统设计的基本语言。
2. 学会一套面试中的沟通和设计框架。
3. 逐个拆解经典系统：限流器、一致性哈希、KV 存储、短链、爬虫、通知、Feed、聊天、搜索补全、视频平台、云盘。
4. 最后整理继续学习的方法。

## 使用提醒

本课程是原创改写和教学化整理，不包含原书长段搬运。建议读者把它当作学习路径和复习材料：先看 lesson，再尝试回答思考题，最后回到自己的项目经验里做迁移。
