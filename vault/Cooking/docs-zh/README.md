# Noma 发酵指南

> *"Our story with fermentation is a story of accidents."* — René Redzepi

这是一门基于René Redzepi和David Zilber的**The Noma Guide to Fermentation**的结构化课程。

## 课程概述

本课程涵盖世界上最具创新性的餐厅之一 Noma 开发和使用的发酵技术。

## 你将学到什么

- **Lacto-fermentation** — 转化水果和蔬菜
- **Kombucha** — 冲泡和调味发酵茶
- **Vinegar** — 从各种来源制作醋
- **Koji** — 培养Aspergillus oryzae
- **Misos and Peaso** — 创建发酵糊状物
- **Shoyu** — 酿造酱油
- **Garum** — 制作古罗马风格的酱汁
- **Black Fruits & Vegetables** — 掌握Maillard反应

*Note: This course is for educational purposes. Always follow proper food safety guidelines when fermenting.*
